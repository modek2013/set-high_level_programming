# Python - Hello World

This project contains introductory Python exercises focused on learning the basics of Python programming, shell scripting, and program execution in a Linux environment.

## Learning Objectives

By completing this project, I aim to understand:

* Why Python is an awesome programming language
* How to run Python scripts
* How to write and execute shell scripts
* How to use Python variables and strings
* Basic Python syntax and coding style
* The purpose of executable files and shebangs

## Project Structure

Each task is implemented in its own file according to the project requirements.

## Requirements

* Ubuntu Linux
* Python 3
* Bash shell
* All scripts are executable
* All Python files begin with:

```python
#!/usr/bin/python3
```

* All shell scripts begin with:

```bash
#!/bin/bash
```
